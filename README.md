# Hidden QR Studio 🚀

An AI-assisted modern QR Code Generator built using pure HTML, CSS and JavaScript.

This project was designed and developed with the help of Artificial Intelligence tools for faster development, smarter UI structuring, feature planning and modern frontend optimization.

The final implementation, customization, branding and deployment were independently managed and refined by the creator.

# ✨ Features

* 15+ QR Code Types
* Real-Time QR Preview
* Full QR Customization
* PNG / SVG / PDF Export
* Offline QR Library
* Analytics Dashboard
* Responsive Modern UI
* No Login Required
* Browser-Based Storage
* Fast & Lightweight
